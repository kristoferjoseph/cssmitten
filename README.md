===========

Customize and generate a simple css style guide.

## install

`npm i cssmitten`

## usage

Customize `colors.js`, `scale.js`, `base.js`, `theme.js`, etc. then generate the `cssmitten.css` file:

`npm run build`

Generate a minified version:

`npm run dist`
